# 九九の練習

## 概要
小学生向けに、九九がランダムに出題されても解けるようにする練習するための小アプリ。

## 公開URL
🌐 **https://toruhjp.github.io/kuku/**

主にタテ向きのスマートフォンやタブレットでの利用を想定する。

## 機能
- 練習したい段（1の段〜9の段）を1つ以上選択できる
- カウントダウン後、ランダムに問題が出題される
- 答えを入力したらすぐに正誤判定される
- 全問回答で完了画面が表示される

## デプロイ
このサイトはGitHub Pagesに自動デプロイされる。

### デプロイフロー
1. `main` ブランチに変更をプッシュ
2. GitHub Actionsが自動的にデプロイ実行
3. https://toruhjp.github.io/kuku/ に反映される
